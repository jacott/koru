Getting started

This guide describes how to build a koru one page application. We will be building a simple todo application called koru-todo.

Create initial System

Ensure you have admin access to postgresql sudo -u postgres createuser -drs $USER

Method one

  • npm install --global koru
  • koru new library
  • cd library

Method two

  • mkdir library && cd library && npm init -y
  • npm install --save koru
  • node_modules/.bin/koru new

Generate a new Model

To generate a model, say, a Book; run the following
  • ./scripts/koru g model Book title author published:date pages:integer
  • Create the database if needed: createdb librarydemo
  • Migrate the database: ./scripts/koru db-migrate
  • Dump datgabase schema to disk: ./scripts/koru db-schema-dump
