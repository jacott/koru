Getting started

This guide describes how to build a koru one page application. We will be building a simple todo application called koru-todo.

Create initial System

Ensure you have admin access to postgresql sudo -u postgres createuser -drs $USER

Method one

  • npm install --global koru
  • koru new koru-todo
  • cd koru-todo

Method two

  • mkdir koru-todo && cd koru-todo && npm init -y
  • npm install --save koru
  • node_modules/.bin/koru new
