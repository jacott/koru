koru/session/publish

Methods

publish.preload = function preload(sub)

If a publish function has a static method called preload it will be called before the request is sent to the server.

Useful actions to take in the preload are (but not limited to):

  1. modifiy the subcription argument list
  2. load data from client-side storage and
  3. call the subscription callback and clear it so not called by server fulfillment
Parameters
subkoru/session/client-sub
Example
const publish = require("koru/session/publish-client");
function publishBooks() {
  v.args = this.args;
}
publishBooks.preload = preload; // preload(sub) {sub.args = [6, 7];}
publish("Books", publishBooks);
subscribe('Books', 5);
assert.equals(v.args, [6, 7]);
